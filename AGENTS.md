# Voltigo Project Guidelines

## Project Overview

Voltigo is a modern web platform for renting eco-friendly vehicles, including electric and hybrid cars.

The goal is to create a **premium, futuristic, and minimal user experience**, inspired by high-end automotive brands and modern startups.

---

## Tech Stack

* Next.js (App Router)
* React
* Tailwind CSS

---

## Design Philosophy

* Clean and minimal UI
* Strong visual hierarchy
* Smooth user experience
* Focus on modern and premium look

---

## Color Direction

The design must follow an **eco-futuristic theme**:

* Primary color: Electric green (#7CFF3C)
* Accent color: Light green (#CFFF1A)
* Background: Dark tones (black / dark green only — NO blue)
* Text: White or light gray

Avoid:

* Bright or saturated unrelated colors
* Blue tones

---

## UI Style

* Dark mode interface
* Soft glow effects (green neon)
* Subtle gradients (dark → darker green)
* Rounded corners (xl or 2xl)
* Clean spacing and alignment

---

## Components Guidelines

All components must be:

* Reusable
* Modular
* Cleanly structured
* Styled using Tailwind only

---

## UX Principles

* Simple navigation
* Clear call-to-actions
* Fast and responsive design
* Mobile-friendly layout

---

## Animations

* Smooth transitions (duration-300)
* Hover effects (scale, glow)
* Subtle fade-in animations

---

## File Structure

* components/
* app/
* services/
* hooks/

Follow a clean and scalable architecture.

---

## Development Rules

* Keep code simple and readable
* Avoid overcomplicating logic
* Use consistent naming conventions
* Separate UI and logic when possible

---

## Goal

Build a professional, modern, and visually impressive web application that showcases eco-friendly mobility in a premium way.
